# ¿Como utilizarlo?

## Paso 1:

Ya sea que clones el repositorio o como obtengas el repositorio tenlo en tu computadora.

## Paso 2:

Lo que nececitas es tener docker, si tienes docker, solo tienes que poner este comando

```bash
docker compose up -d
```

o

```bash
docker-compose up -d
```

## Paso 3

Puedes utilizar la URL http://localhost/api/v1/saludo y solamente recargas la pagina web para notar el cambio entre replicas.
